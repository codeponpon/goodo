# Todo app using Backbone.js, RequireJS, Bootstrap, Yeoman (Grunt + Bower) #

## Install and run ##

	git clone git@github.com:jesucarr/Yeoman-TodoMVC.git
	npm install
	bower install
	grunt server

## Run example test ##
Command line

    grunt test

In browser

    grunt serve:test
